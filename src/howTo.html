How to Whitelist

For the projects
1. Connect your wallet
2. Enter your project name
3. Enter Whitelist Token Address
Create Whitelist Project

On the dashboard you will find
Project name
Whitelist Token Deposit Address
Custom Whitelist Project URL

To utilise this tool:
1. Send tokens to your Whitelist Token Deposit Address
3. You can then share the whitelist URL to your community and whomever uses this link will be able to claim a token
As this is version 1 there are no restrictions to users who have a link and can claim a token. With this in mind it's best to send only what you want to give away at that time.

The best method would be to have your dev create a surplus of WL tokens and divide them to the team. The team can then send WL tokens to the WL wallet and share the link.
Whitelist Token
First you need to install Solana CLI, find out how here - docs.solana.com
Once complete, create your SPL Token using this link - Moralis.io

You've now created your own SPL Token. All thats needed now is to add the WL Token address to your CMv2

For further information on how to use Candy Machine v2 and whitelist please visit:
docs.metaplex.com
For the Users
Follow the URL link provided by the project
Connect your wallet
Select Claim Token
Success you now have the whitelist token to mint your favourite project

You can only claim 1 Whitelist Token per wallet
